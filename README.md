## 🧠 Concepts I'll Cover

<details>
<summary><strong>⚛️ React Fundamentals</strong></summary>

- JSX syntax and rules
- Components, Props
- Rendering lists and conditional rendering
- Component composition and reusability

</details>

<details>
<summary><strong>🔄 State & Events</strong></summary>

- `useState` and controlled components
- Event handling
- Lifting state up
- Derived state

</details>

<details>
<summary><strong>🎣 Hooks Deep Dive</strong></summary>

- `useEffect` & the component lifecycle
- `useRef` for DOM access and persisting values
- `useReducer` for complex state
- `useContext` for global state
- Building custom hooks

</details>

<details>
<summary><strong>🧭 React Router v6</strong></summary>

- Declarative routing
- Dynamic & nested routes
- `useNavigate`, `useParams`, `useSearchParams`
- Protected routes

</details>

<details>
<summary><strong>🏪 Redux & Redux Toolkit</strong></summary>

- Actions, reducers, store
- `createSlice` and `configureStore`
- `useSelector` and `useDispatch`
- RTK Query for data fetching

</details>

<details>
<summary><strong>⚡ Next.js 14</strong></summary>

- App Router & file-based routing
- Server Components vs Client Components
- Server Actions & data mutations
- Authentication with NextAuth
- Optimistic UI updates

</details>
